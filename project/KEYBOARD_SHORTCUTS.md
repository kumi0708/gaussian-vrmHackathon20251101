# 🎹 Gaussian-VRM キーボードショートカット

## 🎮 基本操作（修飾キーなし）

### エフェクトモード切り替え (ModelEffectController)
- **Q**: Bass mode (低音強調)
- **W**: Treble mode (高音強調)  
- **E**: Balance mode (バランス)
- **T**: Punch mode (パンチ)

### カメラ制御
- **W/A/S/D**: カメラ回転 (上下左右)
- **U/O**: カメラズーム (ズームイン/アウト)
- **Z**: アニメーション速度ダウン
- **X**: アニメーション速度アップ

### モデル切り替え  
- **Page Up**: 次のモデル
- **Page Down**: 前のモデル

### アニメーション切り替え
- **テンキー 0-9**: 基本アニメーション
  - 0: Idle, 1: Walking, 2: Breathing, 3: Gangnam Style
  - 4: Warrior, 5: Listening, 6: Shrugging, 7: Chicken Dance
  - 8: Pointing, 9: Around
- **Shift + テンキー**: 追加アニメーション
  - Shift + .: Acknowledging
  - Shift + +: Dizzy Idle
  - Shift + -: Happy Idle
  - Shift + *: Jab Cross

## 🎨 ポストエフェクト（数字キー）

### エフェクト切り替え (PostEffectController)
- **5**: Bloom エフェクト
- **6**: Film エフェクト
- **7**: Glitch エフェクト
- **8**: Scanline エフェクト
- **9**: ColorShift エフェクト
- **0**: エフェクトなし

### エフェクト制御
- **[**: 次のエフェクト
- **]**: ランダムエフェクト
- **\\**: 自動エフェクト切り替えオン/オフ

## 🔊 音声制御 (Ctrl + キー)

### 音量制御
- **Ctrl + M**: ミュート切り替え
- **Ctrl + +**: 音量アップ
- **Ctrl + -**: 音量ダウン
- **Ctrl + 0**: 音量リセット

## 💡 使用のコツ

1. **キー配置の論理**:
   - **Q-T**: オーディオエフェクトモード（Bass, Treble, Balance, Punch）
   - **5-9,0**: ビジュアルエフェクト（Bloom, Film, Glitch等）
   - **WASD**: カメラ回転（上下左右）
   - **U/O**: カメラズーム（イン/アウト）
   - **テンキー**: アニメーション切り替え
   - **Ctrl**: 音量制御のみ
   - **Shift**: テンキーと組み合わせて追加アニメーション

2. **競合なしの設計**:
   - オーディオモード（Q-T）とビジュアルエフェクト（5-9,0）は完全分離
   - カメラ操作（WASD+UO）は専用キー
   - 1-4番キーは現在未使用（将来の拡張用）

3. **よく使う組み合わせ**:
   - **Q** → **5**: Bass強調モード + Bloomエフェクト
   - **Page Up** → **テンキー3** → **7**: モデル切り替え → Gangnam Style → Glitch
   - **U/O** → **WASD**: ズーム調整 → カメラアングル調整

4. **ライブパフォーマンス**:
   - **\\**: 自動エフェクトモードで放置
   - **Page Up/Down**: モデルを次々切り替え  
   - **Q-T**: 音楽に合わせてモード切り替え
   - **[ ]**: エフェクトを手動でサイクル
